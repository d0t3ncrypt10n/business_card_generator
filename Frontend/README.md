# Business Card Generator - Frontend

React frontend for the Digital Business Card Generator.

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Open http://localhost:5173

## Build for Production

```bash
npm run build
```

## Pages

- **Home** - Landing page with features
- **Login/Register** - Authentication
- **Dashboard** - Manage cards
- **Create Card** - Create new business card
- **Edit Card** - Edit existing card
- **Public Card** - View/share card publicly

## Key Dependencies

- React Router DOM - Routing
- Axios - API calls
- qrcode.react - QR code generation
- html2canvas - Card to image conversion
- jsPDF - PDF generation
